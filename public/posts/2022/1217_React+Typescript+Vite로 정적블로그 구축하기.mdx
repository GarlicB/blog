# React + Typescript + Vite로 정적블로그 구축하기

## 1. 블로그를 구축하기까지

많은 개발자들이 커리어를 위해 열심히 공부하고 배운 것들을 블로그에 남기는 것에
감명받아 언제나 그랬지만 늦게나마 기술블로그를 만들었다.

동시에 Github를 이력관리를 위한 저장소 용도로만 사용해왔는데,
정적 페이지 서비스를 Github Pages에 올려 블로그를 운영하기로 했다.

## 2. 구축환경

- ReactJS
- Typescript
- Vite
- Mdx2.0
- Github Pages
- Gihtub Actions
